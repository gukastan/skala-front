# 스토리 구조

## 프롤로그

가상 교육생 김선경이 SKALA에 입과한 직후 중앙 학습 시스템에 오류가 발생한다.
수료 심사에 필요한 네 가지 증빙이 서로 다른 학습관에 분리된다.

## 네 가지 증빙

- STRUCTURE: HTML로 정보를 의미 있게 구조화했는가
- DESIGN: CSS로 화면을 배치하고 표현했는가
- LOGIC: JavaScript로 동작과 데이터 흐름을 구현했는가
- RECORD: Git과 AI 활용 과정을 기록했는가

## 학습관

- HTML 구조관: 프로필·시간표·계획·미디어·폼 복구
- CSS 디자인관: Box Model·Flexbox·Grid·반응형 복구
- JavaScript 보안실: 비밀번호·성적·객체·날씨 작동
- Git·AI 기록관: 커밋과 AI 협업 과정 복원

## 최종 심사

네 개의 증빙은 수료증이 아니라 최종 심사 자격이다.
최종 심사실에서 필수 기능, 설명 가능성, Git 기록, Public 저장소를 점검한 뒤 수료 판정을 받는다.
